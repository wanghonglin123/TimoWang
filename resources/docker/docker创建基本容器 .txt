1：docker pull phusion/baseimage 下载phusion/baseimage镜像

2：sudo docker run -d --name redis -p 50001:22 phusion/baseimage /sbin/my_init --enable-insecure-key 启动phusion/baseimage镜像

3：sudo docker inspect redis | grep IPA 查看运行phusion/baseimage容器Ip

4: apt-get update 更新apt-get
3: apt install net-tools 安装系统命令
4: apt install iputils-ping	ping命令
6: apt-get install openssh-server
7: service ssh start
8: sudo docker stop 容器id 停止容器

9：sudo docker run -d --name redis -p 50001:22 phusion/baseimage /sbin/my_init --enable-insecure-key 
	重启
